====================================================
  AI INFLUENCER STUDIO — SETUP GUIDE
====================================================

Follow these steps in order. No tech experience needed.
Just type exactly what you see in the boxes.


STEP 1 — DOWNLOAD THE PROJECT
--------------------------------------------------
1. Go to: https://github.com/DaanKieft/ai-influencer
2. Click the green "Code" button
3. Click "Download ZIP"
4. Find the downloaded ZIP (usually in your Downloads folder)
   and double-click it to unzip
5. Move the unzipped folder somewhere easy, like your Desktop


STEP 2 — INSTALL ANTIGRAVITY
--------------------------------------------------
1. Search "Antigravity" on Google, or go to: https://antigravity.dev
2. Download it and install it like any normal app

(That's the only thing you install. Everything else is inside it.)


STEP 3 — OPEN THE PROJECT
--------------------------------------------------
1. Open Antigravity
2. Click: File → Open Folder
3. Pick the unzipped project folder from Step 1


STEP 4 — ADD CLAUDE (your helper)
--------------------------------------------------
1. Click the Extensions icon in the left sidebar
   (it looks like four little squares)
2. Search for "Claude Code"
3. Click Install
4. Sign in with your Anthropic account when it asks


STEP 5 — START IT
--------------------------------------------------
1. Open the terminal: Menu → Terminal → New Terminal
2. Type this and press Enter:

      claude

3. When Claude is ready, just tell it:

      install everything and start the app

   Claude does the rest. When it's running, it will show a web
   address that looks like:

      http://localhost:5173

   The number at the end may be different on your computer —
   just use whatever address Claude shows. Open your browser
   and go to that address.

Done! The app loads with everything ready.


STEP 6 — CONNECT HIGGSFIELD (for images & videos)
--------------------------------------------------
1. In the app, click the gear icon (top right) → Settings
2. Click "Connect Higgsfield"
3. Log in with your Higgsfield account
4. Done — it uses your own Higgsfield credits


USING IT LATER
--------------------------------------------------
To start the app again:
   Open Antigravity → open the terminal → type  claude
   → tell it "start the app"  →  open the web address it shows

To stop the app:
   Click the terminal and press Ctrl+C

Keep Antigravity open while using the app — the terminal
inside it is what runs the app.


GOOD TO KNOW
--------------------------------------------------
- You only install Antigravity. It already includes everything else.
- Stuck on anything? Just ask Claude in the terminal. That's what
  it's there for — type your question and press Enter.
- All your data is saved in your browser. Clearing your browser
  data will reset everything.
